# GetClassType

Returns a type of the ApiComplexForm class.

## Syntax

expression.GetClassType();

`expression` - A variable that represents a [ApiComplexForm](../ApiComplexForm.md) class.

## Parametrs

This method doesn't have any parameters.

## Returns

String
