# GetClassType

Returns a type of the ApiUnsupported class.

## Syntax

expression.GetClassType();

`expression` - A variable that represents a [ApiUnsupported](../ApiUnsupported.md) class.

## Parametrs

This method doesn't have any parameters.

## Returns

String
