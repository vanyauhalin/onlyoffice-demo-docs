Form type. The available form types.

type

## Properties

- "textForm" 
- "comboBoxForm" 
- "dropDownForm" 
- "checkBoxForm" 
- "radioButtonForm"

## Example

This example returns a type of the current form.

```javascript
var sFormType = oTextForm.GetFormType();
```
