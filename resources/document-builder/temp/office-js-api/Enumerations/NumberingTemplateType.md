# NumberingTemplateType

The predefined numbering template.

## Type

String

## Properties

- "none" 
- "bullet" 
- "1)" 
- "1." 
- "I." 
- "A." 
- "a)" 
- "a." 
- "i."

## Example

This example sets one of the existing predefined numbering templates.

```javascript
oNumLvl.SetTemplateType("A.");
```
