# highlightColor

Available highlight colors.

## Type

String

## Properties

- "black" 
- "blue" 
- "cyan" 
- "green" 
- "magenta" 
- "red" 
- "yellow" 
- "white" 
- "darkBlue" 
- "darkCyan" 
- "darkGreen" 
- "darkMagenta" 
- "darkRed" 
- "darkYellow" 
- "darkGray" 
- "lightGray" 
- "none"

## Example

This example sets a paragraph highlight in green.

```javascript
oParagraph.SetHighlight("green");
```
